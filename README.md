# my-claude-settings

Personal Claude Code settings and commands.

## Commands

| Command | Description |
|---------|-------------|
| `/reflect` | Session summary + Claude's honest diary |

## Setup

Clone this repo as your Claude Code project settings directory, or copy the `.claude/` folder into your project.

## Settings

- **Default mode**: `plan` — switches to ask/auto for implementation
- **Model**: `opusplan`
